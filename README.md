# EngineerCMS-HanLPService

#### 项目介绍
Springboot搭建

#### 使用
1. 下载后解压

2. 打开cmd

3. cd 到解压路径/EngineerCMS-HanLPService目录下,如（F:\EngineerCMS-HanLPService）

4. 输入命令 

   ```
   mvn package
   ```

5. 输入命令：

   ```
   java -jar ./target/hanlp-service-0.0.1-SNAPSHOT.jar
   ```

6. 浏览器打开swagger查看API

   ```
   http://localhost:8888/swagger-ui.html
   ```
